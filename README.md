# AI Smart Allocation Engine

AI-Based Smart Allocation Engine for PM Internship Scheme.

This system automatically matches students with internships
based on:

- Skills
- CGPA
- Preferences
- Matching score

## Features

- Student Registration
- Internship Management
- AI Matching Engine
- Smart Allocation System
- Ranking System

## Technologies Used

Frontend:
- React
- TypeScript

Backend:
- Node.js

Database:
- SQLite

AI:
- Gemini API

## Run Locally

Install dependencies:

npm install

Start backend:

npx tsx server.ts

Start frontend:

npm run dev

Open browser:

http://localhost:5173
